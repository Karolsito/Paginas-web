Declaraciones en bloque definidas entre etiquetas style

h1 {
  font-size: 2em;
  color: darkcyan;
  text-align: center;
  background-color: yellow;
}

Todos los textos con etiquetas h1 tienen este estilo.

La etiqueta h1 ocupa todo el ancho del contenedor (por default lleva display de bloque).

La etiqueta h1 lleva texto en color cian oscuro.

El fondo de la etiqueta h1 lleva color amarillo.

El estilo definido para la etiqueta h1 indica que el texto irá centrado.

p {
  text-align: left;
  font-size: 1em;
  color: darkblue;
  font-style: oblique;
  border-top: 2px solid gray;
  border-bottom: 2px solid lightgreen;
}

Todos los textos con etiquetas p tienen este estilo definido dentro de la etiqueta style.

Para los párrafos se define un estilo con alineación a la izquierda.

Tamaño de letra de medida 1em.

Color de letra azul oscuro.

Estilo de la fuente oblicuo.

Borde superior sólido de color gris y borde inferior sólido de color verde claro.
