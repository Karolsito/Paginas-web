Preguntas sobre HTTP

  • ¿Sobre qué protocolo binario está montado el protocolo HTTP?

    Este protocolo está montado sobre el protocolo TCP.

  • ¿Cuáles son los clientes http y los servidores http más utilizados?

    Los clientes http más utilizados son navegadores web, CURL y librerías nativas de diversos lenguajes de desarrollo; y los servidores http más utilizados son Apache, Nginx, Tomcat y Microsoft IIS.
